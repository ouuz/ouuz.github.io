安装Node.js和npm

下载Node.js

到Node.js官网进行下载，电脑啥版本下啥版本！

检测Node.js和npm

​ 按住键盘上Shift键的同时，鼠标右击，调出 Power Shell，输入 node -v【-v 前有空格哈】

​ 新版的nodejs已经集成了npm，所以之前npm也一并安装好了。所以我们接着输入 npm -v

​ 输入这两行应该会返回给你一串数字，是版本号哈，不用在意，要是是别的，那说明你就没装好， 老老实实再下载安装一遍就ok Σ(っ °Д °;)っ

以上步骤都没问题的话 QAQ， 安装的环境已经全部搭建完成啦

绑定域名

虽然现在已经可以访问我们的网站，但是网址是GitHub提供的:http://xxxx.github.io ， 而我们想使用我们自己的个性化域名，这就需要绑定我们自己的域名。

先去买一个域名！

​ 我是在阿里云买的，别的网站也行噶，都可以。

​ 接下来演示的是在阿里云购买的，其他的应该大同小异。

​ 打开域名查询，查一下你想买的域名，挑选好后进行购买，购买时会需要你填写一个信息模板并实名认证，照着网站上给的提示一步步做即可。

回到GitHub博客仓库

  • 点击Create new file ，文件名：CNAME

  • 输入内容：你申请下来的域名

  • 保存

域名解析

​ 购买完成后 ，

  • 进入域名控制台中的域名列表，
  • 这时候你会看到一个你已经购买的域名，在右侧有一个解析的按钮 ，点！
  • 添加记录
  • 记录类型：CNAME
  • 主机记录：www
  • 解析线路：默认
  • 记录值：用户名.github.io
  • TTL：不填

看看我的网站？

​ 等个十分钟吧？打开浏览器，输入你刚刚买下的域名，这时候你就可以看到你的博客啦٩(ˊ〇ˋ*)و~
