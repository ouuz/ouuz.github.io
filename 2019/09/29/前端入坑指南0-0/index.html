关于怎么调教老婆（特指 vscode

第一步，装好并打开 vscode

第二步，点开 vscode 左侧列表的插件列表

大家打开的跟我的页面可能有点差别…差在哪里呢？

👧的是中文的，你们的是英文的啊哈哈哈哈

别着急！马上就改成中文的嗯嗯嗯嗯嗯！

点开上面箭头指的按钮后，

在这里！开始复制粘贴！！

这是应用商店，你可以在这里搜索安装好用的程序扩展

我也在这里给大家推荐了十个扩展，先装上！

后面大家的 老婆 （不是）编辑器变成什么样子就看自己了👻

1. Chinese

汉化你的编辑器（英语是阻碍我写代码的第一道门槛…

装好以后还没完！！！同时按下 ctrl+shift+P 调出命令窗口

（命令窗口在顶部顶部 弹出来的一个小输入框）

输入 display 找到图上这个一长串东西然后点进去更改选中 zh-cn

就更改完成啦！

（Tips：如果完成上述操作还是英文的话，关闭编辑器重启试试噢

2. Auto Close Tag

自动闭合标签（也是一个拿来偷懒的插件 QAQ 标签有前后的🐎，装了这个以后，写好前面的，vscode 会帮忙自动补全后面的闭合标签滴

3. Auto Rename tag

自动重新命名标签（比如我本来写的 p 标签，可我想改成 span 了QAQ，可是标签前后都要改，好麻烦，想偷懒…用这个插件就可以同步改好前后啦~！

4. Open in Browser

允许您在默认浏览器中打开当前文件。

5. Live Server

为静态和动态页面热更新

（插话🙃： 虽然好像在写静态页面的时候两个插件差不多，甚至第一个更方便，一个 alt+B 直接就过去了 但…装吧装吧…全装上吧）

6. Prettier

代码格式化工具（一键调好全部缩进还要运算符间隔之类的！妈妈再也不用担心我写的代码密密麻麻被人打😭了！

7. Path Intellisense

文件路径补全，在引进外部样式和逻辑代码文件的时候有这个超级爽😎

8. Code Runner

跑一些简单代码，比如 js、C 等等都可以噢！

9. Color Highlight

颜色代码高亮（这样在编辑器就可以预览颜色，才不要一直重新打开浏览器🙉

10. CSS Peek

小🖱一滑就可以查看你写在这个元素上 HTML + CSS 的全部定义啦

还有个不是插件 但也是 偷懒 （不是）调试代码方便的操作：自动保存

手动放大加粗以表示其重要性！！！

在 vscode 最顶部找到文件（也就是第一个）（就是我箭头画了一堆指的那里！！）

点开！看到那下面的自动保存四个大字了咩！

勾选就完了！一步到位！！！！

从此不再天天 CTRL + S CTRL + S CTRL + S CTRL + S CTRL + S……🤣
